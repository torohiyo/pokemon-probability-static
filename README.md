# ポケカ山札ドロー確率計算

モバイル利用前提の静的Webアプリです。依存関係なしで、GitHub Pagesにそのまま公開できます。

## 使い方

```bash
python3 -m http.server 5173
```

ブラウザで `http://localhost:5173` を開きます。

## 主な機能

- 初期山札枚数・手札枚数の設定
- A/B/Cの目的カード枚数の設定
- 連続プレイ全体での成功率計算
- 縦引き / 山札に戻してシャッフル / 山札の下に戻す / 山札圧縮
- 設定タブからカード追加
- 設定タブからカード効果の並び替え
- カスタムカードと並び順を localStorage に保存
