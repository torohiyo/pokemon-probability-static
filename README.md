# ポケカ山札ドロー確率計算アプリ

モバイル利用を前提にした、ポケモンカード向けの確率計算Webアプリです。依存関係なしの静的HTML/CSS/JavaScriptなので、GitHub Pagesや任意の静的ホスティングにそのままアップできます。

## できること

- 初期状態は「山札枚数」と「手札枚数」だけで設定
- カード画像をタップして、博士・リーリエ・さかてにとる等を順番にシナリオへ追加
- 各ステップごとに「引きたいカード名」と「山に残っている枚数」を設定
- 条件は「どれか1種」「すべて」「合計n枚以上」から選択
- 合計n枚以上は1〜8のボタンで選択
- 全ステップを通した合計成功率をページ上部・下部に表示
- 山札圧縮にも対応

## 登録済みカード効果

- 博士の研究：手札をトラッシュして7枚引く
- リーリエの決心強：手札を山札に戻し8枚引く
- リーリエの決心弱：手札を山札に戻し6枚引く
- スペシャルレッドカード：手札を山札の下に戻し3枚引く
- さかてにとる：山札を3枚引く
- お使いダッシュ：山札を2枚引く
- みどりのまい：山札を1枚引く
- 山札圧縮：山札の枚数をn枚減らす

## 画像URLの差し替え

`assets/app.js` の `EFFECTS` にある `imageUrl` を差し替えるだけで、カード画像が表示されます。

## ローカルで開く

```bash
python3 -m http.server 5173
```

その後、ブラウザで `http://localhost:5173` を開いてください。

## GitHub Pagesへのアップロード

```bash
git init
git add .
git commit -m "Initial probability app"
git branch -M main
git remote add origin <YOUR_REPOSITORY_URL>
git push -u origin main
```

GitHubのRepository Settingsから Pages を開き、`main` branch / root を指定すると公開できます。

## 注意

この版では、各ステップの「山に残っている枚数」は、その効果を使う時点での想定枚数として入力します。前のステップで偶然引いてしまう可能性まで厳密に追跡したい場合は、全カードを共通ターゲットとして管理する拡張が必要です。
