# ポケカ山札ドロー確率計算

スマホ利用前提の静的Webアプリです。依存関係なしで、GitHub Pagesにそのまま置けます。

## 使い方

1. 初期状態として山札枚数・手札枚数を入力
2. 引きたいカードA/B/Cの山札内枚数を入力
3. 詳細条件で「どれか1種」「すべて」「合計n枚以上」を選択
4. カード画像をプレイ順にタップ
5. 上部・下部に全体成功率が表示されます

## 対応効果

- 博士の研究：手札をトラッシュして7枚引く
- リーリエの決心強：手札を山札に戻し8枚引く
- リーリエの決心弱：手札を山札に戻し6枚引く
- スペシャルレッドカード：手札を山札の下に戻し3枚引く
- さかてにとる：山札を3枚引く
- お使いダッシュ：山札を2枚引く
- みどりのまい：山札を1枚引く
- 山札圧縮：目的カード以外をn枚減らす

```bash
python3 -m http.server 5173
```
